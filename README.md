## Setup

This is a static HTML site. Open `index.html` in any browser, or serve with:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`

## Tech Stack

- CSS3 (`style.css`)
- Vanilla JavaScript (`script.js`)
